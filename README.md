# AI训练师（三级）刷题系统

人工智能训练师职业技能等级认定（三级）理论知识刷题工具。

## 功能

- 900+道题库，支持判断/单选/多选
- 自动判题 + 答案解析
- 错题本收藏与重做
- 题号导航面板
- 搜索题目
- 收藏功能
- 每日统计趋势
- 数据导入导出
- 深色/浅色主题切换
- 答题进度自动保存

## 本地运行

```bash
cd quiz-app
python3 -m http.server 8080
```

访问 http://localhost:8080

## 部署

支持 Vercel 一键部署（静态站点）。

## 题库来源

《人工智能训练师（三级）理论知识复习题》—— 上海交通大学职业技能等级认定项目。
